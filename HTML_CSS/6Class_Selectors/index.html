Aula 13- Seletor Before/After

Aqui resumidamente a gente consegue criar um elemento, antes de depois de algum elemento criado dentro do html
por exemplo crio uma div e quero que tenha um quadrado antes dela.

entao abrimos nosso css e colocamos div::before ou div::after
e sempre tem que acompanhar o content

Se reparar criamos uma setinha com a cor azul de fundo

    
        div::after ou before {
            content:"->";
            background-color: blue;
        }

mas digamos que agora vc queira criar um quadrado de fato, quando vc tentar fazer n vai aparecer
porque o after e before e em linha, ou seja para isso vc precisara colocar um display:inline-block;

    
        div::after ou before {
            content:"";
            display:inline-block;
            width:100px;
            height:100px;
            background-color: blue;
        }
