Aula 6-Css @media: Como funciona 1

Aqui conseguimos criar regras para nosso site, quando ele atingir determinados parametros que nos estabelecemos, e como se criassemos
outra folha de CSS, para determinado tamanho do site.

        
            Aqui esta uma configurcao do nosso body normal:
            body {
                background-color:gray;
            }
            
            aqui com esse comando, nosso body muda quando nossa tela atinge determinado tamanho.
            @media (max-width: 600px) {
            
              body {
                    background-color:red;
                }
             
            }

Aula 9-Aspect-ratio no @media

e uma tecnica pouco utilizada

    
        nesse caso quando a tela tiver 1 para 1 ou seja um quadrado, ela tera 
        esses atributos:
        @media (aspect-ratio: 1/1) {
            body {
                background-color: red;
            }
        }
        ja aqui em 2 para 1, no caso quando ela tiver o dobro do tamanho da largura, ela tera esses atributos.
        @media (aspect-ratio: 2/1) {
            body {
                background-color: rgb(0, 110, 255);
            }
        }
        ja aqui e 1 para 2, no caso quando ela tiver o dobro do tamanho da altura, ela tera esses atributos.
        @media (aspect-ratio: 1/2) {
            body {
                background-color: rgb(0, 255, 47);
            }
        }

Aula 13-Funcao max e min

Entao em resumo aqui, conseguimos definir o tamanho da imagem de acordo como tamanho da tela que tivermos por exemplo

    
        img {
            width: max (50%, 400px);
        
        }

No min e o inverso, por exemplo os 50% sera quando estiver abaixo do valor min de pixel que voce definiu.

    
        img {
            width: min (50%, 600px)
        }

Aula 14-Funcao calc

essa funcao calcula tamanhos, digamos que colocamos um width em porcentagem, mas queremos na verdade um espaco de 20px apenas, se deixar em porcentagem quando diminuir o tamanho da tela, consequentemente aquele tamanho tambem ira alterar, porque sera proporcional ao tam da tela aquela porcentagem que definimos 

    
        Entao esse valor de calc basicamente, define a responsividade daquele elemente, por exemplo quando eu for diminuindo a tela, sempre ficara aquele espaco de 40px definido.
        
        .container {
            width: calc(100% - 40px);
            background-color: grey;
            height: 150px;
        }
